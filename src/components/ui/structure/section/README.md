# Section Component

Use Section to render various section components.

## API

- is: type of pattern [less important]
  - content
  - feature
  - form
  - heel
  - hero
- of: type of utility patterns [important]
  - compact
  - contained
  - fluid
  - full
- on: type of parent pattern [more important]
  - composition
    - hero
  - layout
    - default
  - page
    - default
    - home
  - template
    - default
