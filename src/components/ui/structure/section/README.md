# Section Component

Use Section to render various section components.

## API

- is: type of pattern
  - content
  - cta
  - error
  - feature
  - form
  - heel
  - hero
  - navbar
- of: type of utility patterns
  - compact
  - contained
  - fluid
  - full
- on: type of parent pattern
